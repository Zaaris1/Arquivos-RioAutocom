# Arquivos RioAutocom

Este repositório foi preparado para hospedar os arquivos de distribuição do pacote de monitoramento RioAutocom.

## Arquivos para colocar na raiz do repositório

Envie estes arquivos pela aba **Code > Add file > Upload files**:

- `README.md`
- `manifest.json`

## Arquivos para publicar em Releases

Crie uma release, por exemplo `v1.0.0`, e envie estes assets:

- `python-installer.exe`
- `connectivity_app.zip`
- `cert_tele_app.zip`
- `Pacote_de_monitoramento_RioAutocom.exe`

## Links esperados pelo manifest

- `https://github.com/Zaaris1/Arquivos-RioAutocom/releases/latest/download/python-installer.exe`
- `https://github.com/Zaaris1/Arquivos-RioAutocom/releases/latest/download/connectivity_app.zip`
- `https://github.com/Zaaris1/Arquivos-RioAutocom/releases/latest/download/cert_tele_app.zip`
- `https://github.com/Zaaris1/Arquivos-RioAutocom/releases/latest/download/Pacote_de_monitoramento_RioAutocom.exe`

## Observação

Este conjunto já deixa o repositório pronto para o próximo passo: adaptar o instalador para baixar os pacotes por link direto usando esse `manifest.json`.
